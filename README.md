# Smart Irrigation AI

Machine learning project focused on predicting irrigation requirements using environmental sensor data.

## Overview

This project explores how machine learning can support smarter irrigation systems and sustainable agriculture.

Environmental variables such as:
- soil moisture,
- rainfall,
- humidity,
- sunlight exposure,
- temperature,
- and NDVI vegetation data

were used to train multiple machine learning models for irrigation prediction.

## Models Used

- Logistic Regression
- Random Forest Classifier
- Gradient Boosting Classifier

## Features

- Smart irrigation prediction
- Feature engineering
- Model comparison
- F1 score evaluation
- Feature importance analysis
- Environmental data visualization

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Future Improvements

Planned future improvements include:
- real-time IoT sensor integration,
- weather forecasting support,
- autonomous AI agents,
- dashboard deployment,
- and advanced agricultural prediction systems.


## Goal

The goal of this project is to explore how machine learning and AI systems can support precision agriculture and smarter environmental resource management.
